> **本文档面向用户**，不涉及内部技术实现细节。所有规划围绕两个核心目标：
> 1. **更顺手** — 让每一次操作都少一步、快一秒
> 2. **更直觉** — 让新用户不看手册也能上手
>
> **平台范围**：当前聚焦 Windows 10/11 (x64)，暂不考虑 macOS 和 Linux。
> **企业能力**：团队共享、RBAC、SSO 等企业级功能暂不在规划范围内，留待未来根据用户反馈决策。

## 后续规划：使用体验与交互优化

> v2.2.2 功能已全面交付。以下规划聚焦于**使用体验、界面操作、人机交互逻辑**三个维度，分为三个优先级阶段推进。
>
> - **P0（近期）**：直接影响日常使用效率的高频痛点
> - **P1（中期）**：提升操作流畅度和信息可读性的增强项
> - **P2（远期）**：锦上添花的细节打磨和高级交互模式

---

### 第一阶段：高频痛点优化（P0）

> **目标**：解决当前界面中影响日常使用效率的核心痛点，让高频操作更快、更直觉。

#### 1.1 侧边栏工具区重构

| 任务 | 用户价值 |
|---|---|
| **导入/导出合并为菜单** | 当前侧边栏「工具」区域列出 20+ 个平铺按钮，导入导出占了大量空间。将所有导入选项合并为「导入」下拉菜单（Bitwarden / KeePass XML / KDBX / 1Password / EnPass / CSV / .env / 加密容器 / Seed），所有导出选项合并为「导出」下拉菜单（Seed / 加密容器 / .env / JSON / CSV / Markdown），大幅减少侧边栏视觉噪音 |
| **工具按钮分组折叠** | 将工具区按「导入导出」「安全检测」「批量操作」「同步」「其他」分组，每组可折叠/展开，默认只展开最常用的「导入导出」组 |
| **侧边栏宽度可调** | 当前侧边栏固定 220px，部分工具按钮文字被截断。改为可拖拽调整宽度，并记忆用户偏好的宽度 |

#### 1.2 条目列表交互增强

| 任务 | 用户价值 |
|---|---|
| **键盘上下导航** | 在条目列表中支持 ↑/↓ 方向键浏览条目，Enter 打开详情，无需鼠标点击。当前必须鼠标点击才能切换条目，键盘用户效率低 |
| **条目列表内联快速复制** | 每个条目行右侧显示主字段的快速复制按钮（如 API Key 条目直接显示复制按钮），无需进入详情面板即可一键复制最常用的字段 |
| **条目类型视觉标识** | 列表项左侧添加类型图标/色标区分条目类型（API Key / SSH Key / 证书 / 笔记 / OAuth），一眼区分不同类型的条目 |
| **列表项右键上下文菜单** | 为条目列表项添加完整的右键菜单（编辑 / 复制 / 克隆 / 删除 / 移动到文件夹 / 添加标签 / 导出 / 查看历史 / 复制字段值），减少操作步数 |

#### 1.3 搜索体验优化

| 任务 | 用户价值 |
|---|---|
| **搜索历史记录** | 搜索框下拉显示最近 10 条搜索记录，点击即可重新搜索。当前每次搜索都需重新输入，重复搜索效率低 |
| **搜索结果高亮** | 搜索结果中匹配的关键词以高亮背景色标记，快速定位匹配位置。当前搜索只过滤结果但不高亮匹配内容 |
| **搜索快捷键 `/`** | 按 `/` 键快速聚焦搜索框（当前搜索框有 `/` 提示但未绑定快捷键），与主流应用一致 |
| **搜索结果计数实时显示** | 搜索时在搜索框右侧实时显示匹配条目数（如 "3 / 128"），无需滚动列表才知道有多少匹配 |

#### 1.4 详情面板优化

| 任务 | 用户价值 |
|---|---|
| **详情面板宽度可调** | 当前详情面板固定 380px，长字段值被截断。改为可拖拽调整宽度，记忆用户偏好 |
| **字段值一键复制优化** | 每个字段行复制按钮增大点击区域，并添加复制成功视觉反馈（按钮短暂变绿 + Toast 提示） |
| **TOTP 验证码常驻显示** | 详情面板中 TOTP 字段始终显示验证码和倒计时进度环，无需点击展开。当前验证码显示不够醒目 |
| **条目操作快捷按钮栏** | 详情面板顶部添加快捷操作栏（编辑 / 复制主字段 / 复制全部 / 导出 / 查看历史），减少右键菜单依赖 |

### 第二阶段：操作流畅度与信息可读性（P1）

> **目标**：提升信息呈现质量和操作流畅度，让界面更清爽、信息更易读。

#### 2.1 编辑器交互优化

| 任务 | 用户价值 |
|---|---|
| **内联编辑模式** | 支持在详情面板中直接点击字段值进行编辑（点击即编辑模式），无需每次都打开编辑器窗口。对于简单字段修改（如更新 API Key 值），减少窗口切换开销 |
| **字段拖拽排序** | 编辑器中的字段支持拖拽重新排序，当前只能通过删除后重新添加来调整顺序 |
| **模板字段预填充** | 选择平台模板后自动填充标准字段名称和类型，用户只需填入值。当前选择模板后字段为空，需要手动添加 |
| **编辑器未保存提醒** | 编辑器中有未保存的修改时，关闭窗口或切换条目时弹出确认对话框，防止意外丢失编辑内容 |
| **密码字段强度实时提示** | 编辑器中输入密文类型字段值时，实时显示密码强度评分（弱/中/强），引导用户使用强密码 |

#### 2.2 设置界面优化

| 任务 | 用户价值 |
|---|---|
| **设置分类标签页** | 当前设置窗口是一个长滚动列表，改为标签页分区（通用 / 安全 / 同步 / 浏览器扩展 / 危险操作），减少滚动查找 |
| **设置项即时生效** | 大部分设置项（主题、语言、自动锁定时间等）修改后即时生效，无需手动保存。当前需点击保存按钮 |
| **设置搜索** | 设置窗口顶部添加搜索框，输入关键词快速定位设置项（如输入 "剪贴板" 直接跳转到剪贴板清空时间设置） |

#### 2.3 同步体验优化

| 任务 | 用户价值 |
|---|---|
| **同步进度条** | 执行 WebDAV/S3 同步时在状态栏显示进度条和预估时间，而非仅显示"同步中"。对于大金库同步，用户需要知道进度 |
| **同步日志可视化** | 同步完成后可查看详细的同步日志（拉取/推送/合并了哪些条目、是否有冲突），当前只显示简单的 Toast 提示 |
| **同步冲突批量处理** | 当出现多个冲突时，提供「全部采用本地」「全部采用远端」「逐个确认」三种批量处理模式，而非逐个弹出冲突对话框 |

#### 2.4 窗口与导航优化

| 任务 | 用户价值 |
|---|---|
| **面板布局记忆** | 记住用户调整后的侧边栏宽度、详情面板宽度、窗口大小，下次启动自动恢复 |
| **详情面板可关闭** | 支持关闭详情面板以获得更宽的条目列表视图，适合在条目较多时浏览。双击条目重新打开详情 |
| **Esc 键统一关闭** | 所有弹出窗口/对话框统一支持 Esc 键关闭，当前部分窗口不支持 |
| **F2 快捷编辑** | 选中条目后按 F2 直接进入编辑模式，与 Windows 资源管理器重命名快捷键一致 |

#### 2.5 视觉与信息呈现

| 任务 | 用户价值 |
|---|---|
| **条目计数角标** | 文件夹、标签、收藏夹等侧边栏筛选项显示对应条目数量，当前部分项已有但标签面板缺失 |
| **过期条目醒目标记** | 列表中过期/即将过期的条目以红色/橙色角标标记，而非仅在详情页显示警告 |
| **空状态引导优化** | 首次打开空金库时显示引导卡片（「新建第一个条目」「从其他密码管理器导入」「配置同步」），而非仅显示空白列表 |
| **Toast 通知堆叠与自动消失** | 多个 Toast 通知垂直堆叠显示，每条 3 秒后自动消失。当前通知可能互相覆盖 |

### 第三阶段：细节打磨与高级交互（P2）

> **目标**：锦上添花的交互细节，让产品从「能用」提升到「好用」。

#### 3.1 高级交互模式

| 任务 | 用户价值 |
|---|---|
| **命令面板（Ctrl+Shift+P）** | 类似 VS Code 的命令面板，输入关键词快速执行任意操作（新建条目 / 切换 Profile / 同步 / 导入 / 导出 / 设置），无需记住菜单位置。适合高级用户 |
| **条目列表虚拟化** | 当条目数量超过 500 时启用虚拟化列表，保证万级条目下流畅滚动。当前条目较多时滚动可能卡顿 |
| **全局搜索面板（Ctrl+Shift+F）** | 弹出全屏搜索面板，支持跨字段、跨 Profile 搜索，搜索结果以卡片网格形式展示，支持直接在搜索结果中复制字段值 |
| **条目拖拽到文件夹** | 支持从条目列表直接拖拽条目到侧边栏文件夹，快速移动归类。当前需在编辑器中选择文件夹 |
| **多标签页视图** | 支持同时打开多个条目详情标签页，方便对比不同条目的信息。当前切换条目即替换详情面板 |

#### 3.2 新手引导与帮助

| 任务 | 用户价值 |
|---|---|
| **首次使用引导流程** | 首次创建金库后显示 3-5 步引导卡片（新建条目 → 选择模板 → 配置同步 → 设置自动锁定 → 了解快捷键），帮助新用户快速上手 |
| **快捷键速查卡** | 按 `?` 或 `F1` 弹出快捷键速查面板，显示所有可用快捷键。当前快捷键分散在手册中，不够方便 |
| **操作提示气泡** | 首次使用某些功能时（如密码生成器、泄露检测、SSH Agent）显示一次性操作提示气泡，引导用户发现功能 |
| **内置帮助中心** | 设置或侧边栏添加「帮助」入口，内嵌常见问题和使用指南，无需跳转外部文档 |

#### 3.3 通知与提醒优化

| 任务 | 用户价值 |
|---|---|
| **通知中心** | 窗口内添加通知中心图标（右上角铃铛），聚合所有未读通知（过期提醒 / 泄露警告 / 同步异常 / 更新可用），当前通知一闪即逝容易错过 |
| **过期提醒分级** | 即将过期（7 天内）显示黄色提醒，已过期显示红色警告，区别对待不同紧急程度 |
| **定期安全健康报告** | 每周/每月生成安全健康报告（弱密码数量 / 泄露密码数量 / 过期条目数量 / 上次同步时间），以通知中心卡片形式展示 |

#### 3.4 辅助功能与可访问性

| 任务 | 用户价值 |
|---|---|
| **高对比度模式** | 提供高对比度主题选项，方便视力障碍用户使用 |
| **键盘全操作支持** | 确保所有功能均可通过键盘完成（Tab 导航 / Enter 确认 / Space 切换 / 方向键选择），不依赖鼠标 |
| **字体大小调节** | 设置中提供字体大小选项（小 / 中 / 大 / 自定义），适应不同屏幕分辨率和视力需求 |
| **屏幕阅读器支持** | 为关键 UI 元素添加 AutomationProperties 辅助标签，支持 Windows Narrator 等屏幕阅读器 |

#### 3.5 细节微优化

| 任务 | 用户价值 |
|---|---|
| **窗口最小宽度保护** | 窗口缩放到很小时侧边栏、列表、详情三栏不挤压重叠，设置合理的各栏最小宽度 |
| **加载状态骨架屏** | 解锁金库、加载条目列表时显示骨架屏占位，而非空白页面，减少感知等待时间 |
| **复制操作声音反馈** |（可选）复制字段值后播放轻微的确认音效，可在设置中开关 |
| **条目列表密度选项** | 提供「紧凑 / 标准 / 宽松」三种行高选项，适应不同使用场景（紧凑适合大量条目浏览，宽松适合触屏） |
| **深色模式过渡动画** | 切换深色/浅色主题时添加平滑过渡动画，当前为瞬间切换 |
