## 近期计划（v1.2，预计 4-6 周）

> **目标**：在现有基础上打磨体验细节，修复已知问题，让日常使用更流畅。

### 1. 更安全

#### 1.1 锁定时彻底清除敏感数据（INV-03）
**用户价值**：锁定保险箱后，确保内存中的所有密码、密钥被彻底清零，不留痕迹。

- 当前：锁定时清除主密钥，但条目字段值可能残留在内存中
- 目标：锁定时遍历所有敏感字段并显式清零
- 预估：2 天

#### 1.2 日志自动脱敏
**用户价值**：防止日志文件意外记录你的密码或密钥。

- 实现 Serilog 日志过滤器，自动识别 `password` / `secret` / `api_key` / `token` 等字段名并替换为 `***`
- 覆盖所有日志输出路径
- 预估：1 天

#### 1.3 主密码强度评估
**用户价值**：创建保险箱或修改密码时，实时告诉你密码有多强，弱密码直接拒绝。

- 引入 zxcvbn 算法评估密码强度（0-4 分）
- 创建保险箱向导 + 修改密码对话框集成实时评估
- 强度 < 3 时拒绝创建，并给出改进建议
- 预估：3 天

#### 1.4 安全不变量测试补全
**用户价值**：用自动化测试证明安全承诺不是空话。

- 锁定后内存扫描测试（验证密钥已清零）
- 崩溃转储不含主密钥测试
- 1 字节篡改拒绝测试
- 同步路径仅密文测试
- 预估：2 天

### 2. 更方便

#### 2.1 主题切换（浅色 / 深色 / 跟随系统）
**用户价值**：让你在不同光线环境下都能舒适使用。

- 已有 `ui.theme` 配置项，但 GUI 尚未完整实现运行时切换
- 实现浅色 / 深色 / 跟随系统三种模式，设置中一键切换
- 预估：3 天

#### 2.2 同步失败友好提示
**用户价值**：自动同步出问题时，让你知道发生了什么，而不是静默失败。

- ✅ **已实现（v1.2）**：同步合并失败时弹出 Toast 提示
- ✅ **已实现（v1.2）**：连续 5 次失败自动暂停同步并提醒用户手动检查
- ✅ **已实现（v1.2）**：按异常类型分类提示：金库不匹配 / 冲突 / 网络问题 / 文件损坏
- ✅ **已实现（v1.2）**：跨设备同步 UUID 校验，空金库自动拉取远端

#### 2.3 搜索优化
**用户价值**：海量条目时搜索更快，结果更精准。

- 构建真正的倒排索引（当前是线性扫描，虽已足够快）
- 搜索框输入加 250ms 防抖，避免每次按键都触发搜索
- 预估：3 天

#### 2.4 启动错误友好提示
**用户价值**：应用启动失败时告诉你原因，而不是闪退。

- 启动异常时弹出错误对话框，显示原因和解决建议
- 当前异常被静默吞噬，用户完全不知道发生了什么
- 预估：2 小时

#### 2.5 消除界面卡顿
**用户价值**：操作流畅，无卡顿、无阻塞。

- 修复 GUI 中 7 处同步等待导致的线程池阻塞
- 修复 GUI 事件处理器泄漏（窗口关闭后定时器仍在运行）
- 预估：1 天

## 中期计划（v1.3 - v1.5，预计 2-3 个月）

> **目标**：扩展使用场景，增强跨平台能力，提升企业可用性。

### 3. 更安全

#### 3.1 凭据泄露检测（HaveIBeenPwned 集成）
**用户价值**：主动检测你的密码或 API 密钥是否已在已知泄露中，提前告警。

- 使用 HIBP k-anonymity 模式（仅传 SHA-1 前 5 字符，不泄露完整密码）
- 启动时检测主密码是否泄露
- 条目字段可选检查
- 密码健康度面板可视化
- 预估：1 周

#### 3.2 SSH 私钥管理
**用户价值**：开发者高频需要管理 SSH 私钥，当前只能管 API key / token。

- 新增 SSH 私钥字段类型
- ssh-agent 集成（Windows OpenSSH Agent / macOS / Linux）
- CLI：`okv ssh add --id <entry>` 一键加载
- GUI：编辑器加"加载到 ssh-agent"按钮
- 预估：1 周

#### 3.3 审计日志（本地）
**用户价值**：让你知道"谁在何时操作了什么"，满足合规需求。

- 记录所有关键操作：解锁 / 锁定 / 创建 / 编辑 / 删除 / 轮换 / 改密 / 同步
- 持久化到本地审计日志文件
- GUI 查看界面（列表 + 筛选 + 导出）
- 预估：1 周

#### 3.4 紧急联系人 / 信任恢复
**用户价值**：忘记主密码 + 丢失 Recovery Key 时的最后兜底。

- 指定 N 个紧急联系人（每人一个公钥）
- Shamir 秘密共享（M-of-N）分片主密钥
- 延时解锁机制（联系人确认后等待可配置天数，期间可取消）
- 预估：3 周

### 4. 更方便

#### 4.1 macOS / Linux 客户端
**用户价值**：跨平台使用，不再局限于 Windows。

- Avalonia UI 70% 代码可复用
- 适配 macOS Keychain / Linux Secret Service 作为设备密钥存储
- 适配 macOS 公证 / Linux AppImage 打包
- 跨平台自动锁屏 + 文件同步
- 预估：12 周（可与下方功能并行）

#### 4.2 浏览器扩展（只读）
**用户价值**：在网页中直接取用 API 密钥，无需切换应用复制粘贴。

- Native Messaging 协议与本地 `okv` 通信
- Chrome / Firefox / Edge 支持
- 只读模式，不提供修改入口
- 用户确认后才输出字段值
- 预估：6 周

#### 4.3 全局热键解锁 + 自动填充
**用户价值**：在任何应用中按热键即可弹出解锁 → 自动填充当前焦点字段。

- Windows：`RegisterHotKey` API
- macOS：`CGEventTap`（需辅助功能权限）
- Linux：X11 `XGrabKey`
- 预估：2 周

#### 4.4 更多导入格式
**用户价值**：方便从其他密码管理工具迁移。

- 1Password `.1pux`
- LastPass CSV
- Chrome / Edge / Firefox 浏览器密码 CSV
- Safari CSV
- 预估：1 周（高优先 3 个格式）

#### 4.5 更多平台模板
**用户价值**：覆盖更多你使用的平台。

- 1Password / Bitwarden / HashiCorp Vault
- Kubernetes Secret / Docker Hub / NPM / PyPI / Cargo / NuGet
- DigitalOcean / Linode / Vultr / Hetzner / Scaleway
- Figma / Notion / Linear
- 预估：2 周（15 个模板，社区可贡献）

#### 4.6 多保险箱支持
**用户价值**：工作与个人凭据完全分离。

- 每个保险箱独立设备密钥
- GUI 保险箱切换器
- 预估：2 周
